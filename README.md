# Atlas Language OS

Offline-first Progressive Web App for English learning (A1-C1) built with React + TypeScript + Vite.

## Features

- Dashboard with streak 🔥, XP, level, mission queue
- Learning path with CEFR progression (A1-A2-B1-B2-C1)
- Vocabulary flashcards with spaced repetition scheduling
- Shadowing studio using Web Speech API + Web Audio API
- Reading mode with click-to-save vocabulary and comprehension prompts
- Offline conversation simulator with 6 scenarios
- Statistics page with Recharts graphs + activity heatmap
- Achievements, mini games, and daily/weekly/monthly quests
- Local persistence via IndexedDB and localStorage
- PWA support through manifest + service worker

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Zustand
- React Router
- Framer Motion
- Recharts
- IndexedDB (`idb`)

## Run

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```
